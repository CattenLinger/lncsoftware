#软件园协会网站

##写点什么做介绍？

给师弟们写个网站什么的 ˊ_>ˋ，练手作品之一

##怎么跑起来？

请先安装 maven， 然后终端里跑起来这句话

    mvn package

好了，在 target 文件夹里面就有 war 包了，自己拿到心意的 servlet 容器里跑起来就好了。

对了别忘记查看 applicationContext.xml 里面有关数据库的配置，配置好链接并创建好数据库。
不需要建表的， Hibernate 会帮你弄好这些琐碎事 ˊ_>ˋ
